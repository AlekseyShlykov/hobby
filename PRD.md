# PRD: Тест для подбора хобби

## 1. Обзор продукта

### 1.1 Цель
Веб-сервис, который за 5-7 минут определяет тип личности пользователя и рекомендует подходящие хобби с объяснением, почему они подойдут.

### 1.2 Ценность для пользователя
- Узнать о себе (профиль OCEAN)
- Получить персонализированные рекомендации хобби
- Понять, почему именно эти хобби подходят

### 1.3 Структура теста
| Блок | Вопросов | Формат |
|------|----------|--------|
| OCEAN (личность) | 10 | Шкала Лайкерта 1-5 |
| Визуальный выбор | 5 | Выбор 1 из 2 картинок |
| RIASEC (интересы) | 10 | Шкала Лайкерта 1-5 |
| Контекст | 3 | Выбор варианта |
| **Итого** | **28** | ~5-7 минут |

## 5. Тексты результатов по OCEAN

### 5.1 Шкала интерпретации

Для каждой черты: сумма баллов 2-10, где:
- 2-4: Низкий уровень
- 5-6: Средний уровень
- 7-10: Высокий уровень

### 5.2 Тексты для Openness (Открытость)

**Высокая O (7-10):**
```
Исследователь новых горизонтов

Ты обладаешь богатым воображением и тягой к новому. Рутина тебя
утомляет — тебе нужны свежие впечатления, нестандартные идеи и
возможность экспериментировать. Ты ценишь искусство, интересуешься
разными культурами и не боишься менять привычный уклад жизни.

Тебе подойдут хобби, где можно творить, исследовать и постоянно
открывать что-то новое.
```

**Средняя O (5-6):**
```
Баланс стабильности и новизны

Ты умеешь сочетать привычное с новым. Тебе интересны свежие идеи,
но ты не бросаешься в них с головой — предпочитаешь сначала
присмотреться. Это помогает находить хобби, которые и увлекают,
и не требуют постоянной перестройки жизни.
```

**Низкая O (2-4):**
```
Ценитель проверенного

Ты предпочитаешь надёжное и понятное. Резкие перемены — не твоё,
а вот глубокое погружение в одну тему приносит удовольствие.
Тебе подойдут хобби с понятной структурой, где можно постепенно
совершенствоваться в чём-то конкретном.
```

и позволить себе спонтанность — в зависимости от настроения

## 7. Алгоритм подбора хобби

### 7.1 Расчёт профиля пользователя

```python
# OCEAN (нормализация 0-1)
ocean = {
    "O": (q1 + q2 + visual_bonus_O) / max_score,
    "C": (q3 + q4 + visual_bonus_C) / max_score,
    "E": (q5 + q6 + visual_bonus_E) / max_score,
    "A": (q7 + q8) / max_score,
    "N": (q9 + q10) / max_score
}

# RIASEC (нормализация 0-1)
riasec = {
    "R": (q11 + q12 + visual_bonus_R) / max_score,
    "I": (q13 + q14 + visual_bonus_I) / max_score,
    "A": (q15 + q16 + visual_bonus_A) / max_score,
    "S": (q17 + q18 + visual_bonus_S) / max_score,
    "E": (q19 + q20 + visual_bonus_E) / max_score,
    "C": (q21 + q22 + visual_bonus_C) / max_score
}
```

### 7.2 Фильтрация по контексту

```python
filtered_hobbies = [
    h for h in hobbies
    if user_time in h.time
    and user_budget >= h.budget
    and (user_activity == "any" or user_activity == h.activity)
]
```

### 7.3 Скоринг хобби

```python
def score_hobby(hobby, user_riasec, user_ocean):
    # Косинусное сходство RIASEC
    riasec_score = cosine_similarity(hobby.riasec, user_riasec)

    # Бонус от OCEAN
    ocean_bonus = sum(
        user_ocean[trait] * boost
        for trait, boost in hobby.ocean_boost.items()
    )

    return riasec_score * 0.7 + ocean_bonus * 0.3
```

### 7.4 Формирование топ-5

```python
recommendations = sorted(
    filtered_hobbies,
    key=lambda h: score_hobby(h, user_riasec, user_ocean),
    reverse=True
)[:5]
```
